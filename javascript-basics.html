Instructions

Make your changes in a new file called script.js:

  1. Create a JavaScript file in your project folder. Link this file to your .html file by adding a script tag in your .html page.
  2. At the top of your JavaScript file, create a constant variable called firstName and set its value to your first name. Next, create another constant variable called lastName and set its value to your last name. Then, log both of them to the console using console.log().
  3. Make a new function called myFullName() that takes two arguments: (1) fName and (2) lName. In the function body, let there be a new variable called fullName that is your first and last names concatenated (put together). Then, log your fullName variable to the console. Finally, call the function, passing in the firstName and lastName variables you previously created as parameters. Remember that a function can be created by using the function keyword, assigning the function a name, and then including parameters inside parentheses (parameter1, parameter2). What the function will actually do is defined between curly brackets { console.log("hello world"); }:
    4. Example:
    function getSumOfTwoNumbers(number1, number2) {
        return number1 + number2
    };
    getSumOfTwoNumbers(4, 5)
  4. Create an array using let that has different animal names in it. Then, use a for loop to log each one to the console.
  5. Create an object called bestFriend that has three fields: firstName, lastName, and age. Then, log it to the console.
  6. Create a variable using let called friends. Let friends be an array of objects with three fields: firstName, lastName, and age. Then, log the friend to the console. Remember that an array is defined using square brackets [0, 1, 2, 3] while an object is defined using curly brackets and key-value pairs { species: "chicken", numberOfFeet: 2 }. Thus, a single friend will look something like this:
    7. Example:
    let friend = { 
        firstName: 'Hayden', 
        lastName: 'Hoopes', 
        age: 23 
    };
  7. Construct a for loop that passes through each friend in your array of friends and logs their first name to the console. Remember that a for loop uses the key word for followed by parentheses (chicken of chickens) which define what the individual array element will be called from the entire array that has already been defined. What the loop actually does will be contained in curly brackets { console.log(chicken.numberOfFeet) }:
    8. Example:
    let arrayOfNumbers = [0, 1, 2, 3];
    let sum = 0;
    for (number of arrayOfNumbers) {
        sum = sum + number;
    }
    console.log(sum);   // sum will equal 6
  8. Define a constant variable called myAge that is set to your current age. Construct a for loop that logs each of your friendss first name and age to the console only if they are younger than you. Do not log the name or age of friends who are your same age or older. Remember that an if statement looks like this:
Example:
if (friend.numberOfChickens != 8) {
    console.log(friend.name + " does not have 8 chickens");
} else {
    console.log(friend.name + " has exactly 8 chickens!");
}
