This is an example

  1. Open up this HTML file in your web browser (ie. Chrome or Firefox). You should see this text displayed.
  2. Inspect the page by right clicking the page and selecting "Inspect" from the menu. You can also press Ctrl+Shift+I.
  3. The screen should have changed and now you should see a new box on the side of your browser. This is the Inspection window.
  4. At the top of the Inspection window you will see a tab called Console. Click on it to open up the console and view the output of your JavaScript console.log() function.
  5. Link a JavaScript file to this HTML page by adding a <script> tag in the head. Click here for more information about linking your .js file.
  6. Follow the instructions given on Canvas to write JavaScript code and observe the results on the Inspection window!
